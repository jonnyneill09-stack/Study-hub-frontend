# Study Hub

Study Hub is a modern, student-friendly web application designed to help students organize and manage their study materials. Built with React and Vite, it features an intuitive single-page application (SPA) interface with smooth navigation and responsive design.

## Features

- **Dashboard:** Overview of your notes, flashcards, and past papers.
- **Notes:** Manage and view your study notes.
- **Flashcards:** Create and review flashcards for effective learning.
- **Past Papers:** Access and download past exam papers.
- **Authentication:** Login and register forms ready for backend integration.
- **Responsive Design:** Mobile-friendly layout with collapsible sidebar.
- **Styled Theme:** Purple navbar (#4f46e5), white text, clean and modern UI.

## Prerequisites

- Node.js v16 or higher recommended
- npm (comes with Node.js)
